# 🌞 Calculadora de Energia Solar

Calculadora para simular economia com energia solar fotovoltaica.

## 🚀 Funcionalidades
- Cálculo de sistema solar personalizado
- Simulação de investimento e payback
- Interface responsiva
- Validação de inputs

## 🛠️ Tecnologias
- Next.js 14
- TypeScript
- Tailwind CSS
