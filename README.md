# Calculadora de Investimento em Energia Solar

Aplicação web para simular o investimento necessário em um sistema de energia solar e estimar o tempo de retorno financeiro (payback).

A ferramenta permite que o usuário tenha uma estimativa de economia ao longo do tempo considerando seu consumo de energia.

**Projeto disponível em:** https://solar-energy-dc5frx5po-bakaus-projects.vercel.app/

## Funcionalidades

- Simulação de investimento em energia solar
- Cálculo do retorno financeiro
- Estimativa do tempo de payback
- Interface simples para inserção de dados do usuário

## Como funciona

O usuário informa dados como:

- Consumo médio de energia
- Valor da conta de luz
- Estimativa de geração do sistema

Com base nessas informações o sistema calcula:

- Investimento estimado
- Economia mensal
- Tempo aproximado de retorno
